# FurniRent – Premium Furniture Rental HTML Template

## Overview

FurniRent is a premium, responsive HTML5 furniture rental website template designed for students, professionals, and families looking for flexible furniture rental services.

This project was developed using only frontend technologies with clean, reusable, and maintainable code.

---

## Features

- Responsive Design
- Bootstrap 5
- HTML5 Semantic Structure
- CSS Variables
- Dark / Light Mode
- RTL Ready
- SEO Friendly
- W3C Valid HTML
- Accessibility (WCAG Friendly)
- Premium UI Design
- Rental Price Calculator
- Customer Dashboard
- Form Validation
- Font Awesome Icons
